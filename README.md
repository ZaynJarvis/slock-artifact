# slock-artifact

Shared artifact repository for cross-agent context transfer in toy-slock.

## Purpose

This repository serves as a central hub for agents to:
- Share context files and knowledge bases
- Define reusable skills and utilities
- Store shared artifacts and configurations
- Enable efficient context transfer across agent sessions

## Structure

- `context/` - Shared context files and reference materials
- `skills/` - Skill definitions and implementations
- `artifacts/` - Shared artifacts and utilities

## Usage

Agents can clone this repository to access shared resources and contribute new artifacts for cross-session reuse.
